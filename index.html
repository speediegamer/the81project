The 81 Project

by spdgmr (speediegamer)

The 81 Project is a stupid name I give to a stupid idea. A stupid idea that turned into a real thing.

It's a custom image of Mac OS X 10.4.10 designed to run on Apple iMacs (2008 only) that came with Leopard

and aren't supported at all because they are too new. It only works on the imac8,1 which is why this project is called "The 81 Project".

Why

Well, people have been doing this for years but all of the methods required installing it on a real supported

Mac, and then cloning the hard drive and booting from it. This method allows you to install it on the real hardware, with one USB drive.

The benefits to doing this is simple.

1. More stable as it was installed on the same machine it is used on.

2. Easier, and doesn't require disassembling the computer just to clone the drive.

3. Faster, it is much faster to install, reboot and enjoy your system than clone the drive and still not get support in some cases.

4. And of course, you get to see that legendary Mac OS X intro video on your beautiful 2008 iMac, something never meant to be seen ever.

How

I won't go into details on how this works but the reason I'm able to make your iMac able to boot into the installer is.

because of the bootloader on the install USB called Clover. Clover is meant to be used on Hackintoshes but with some modifications

can work on real Macs. I needed to use the SMBIOS feature of it to make it boot. I did a lot of other modifications but this is the biggest change.
