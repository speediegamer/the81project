The NEW 81 Project

by spdgmr (speediegamer)

The 81 Project started as a dumb idea and it actually became a real thing.

It's a custom image of Mac OS X 10.4.11 designed to run on Apple branded computers from 2008 through maybe 2009.

These Macs usually come preinstalled with Leopard but sometimes Snow Leopard.

About

The name is because it was originally made for a early 2008 iMac which has the SMBIOS, imac8,1 aka the name.

Regardless, this was not possible before and as far as I know, I might be the first person to access the setup assistent on a 2008 Mac.

It originally only worked on 2008 iMacs but I have since expanded it to work on some models.. However driver support is not guaranteed. Always tinker at your own risk.

It IS possible to install Tiger on a 2008 Mac before this however this simplifies the process and is so far the only way to see the intro video and install it natively.

In fact, it is incredibly easy to use.. On Windows, you can use TransMac to restore the DMG to a 16GB flash drive.. It needs to be 16GB or larger as the DMG is slightly larger than 14GB in size.

For Apple Macs, you can use Etcher or Disk Utility which is built in macOS although I haven't tried it.

There will be two EFI boot options. If you see the Apple Logo, you booted into the wrong volume.. You should see a boot picker like screen. After, all you have to do is press enter once.

How does this work?

I won't go into details on how this works but the reason I'm able to make your Mac able to boot into the installer is.

because of the bootloader on the install USB called Clover. Clover is meant to be used on Hackintoshes but with some modifications

can work on real Macs. I needed to use the SMBIOS feature of it to make it boot. I did a lot of other modifications but this is the biggest change.
